## 第四步：进阶用法——流式输出和推理内容

DeepSeek R1 支持流式输出，这对实时对话或长文本生成很有用。把上面的 `stream` 改为 `True`，然后迭代获取内容：

python
# 流式调用示例
stream = client.chat.completions.create(
    model="deepseek-reasoner",
    messages=[
        {"role": "user", "content": "解释一下相对论的基本原理"}
    ],
    stream=True
)

for chunk in stream:
    if chunk.choices[0].delta.content:
        print(chunk.choices[0].delta.content, end="")


DeepSeek R1 还有一个独特特性：它会返回推理过程中的中间思考内容，它是一个特殊的 `reasoning_content` 字段。如果你想捕获这个内容，可以这样修改代码：

python
response = client.chat.completions.create(
    model="deepseek-reasoner",
    messages=[{"role": "user", "content": "1+1为什么等于2？"}],
    stream=False
)

# 打印完整的响应数据结构
print(response)
# 或者定位到 reasoning_content
if hasattr(response.choices[0], 'message') and hasattr(response.choices[0].message, 'reasoning_content'):
    print("推理过程：", response.choices[0].message.reasoning_content)


这个特性是 R1 模型独有的，非常强大，适合用在需要展示模型推理过程的场景中，比如教育、复杂逻辑推演。
