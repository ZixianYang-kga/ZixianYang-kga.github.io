我们直接上代码。下面这个示例，我以Java项目中最常用的OkHttp + 原生OpenAI SDK来演示。你只要确保项目里导入了OpenAI的Java包。

java
import com.openai.client.OpenAIClient;
import com.openai.client.okhttp.OpenAIHttpClient;
import com.openai.models.chat.completions.ChatCompletion;
import com.openai.models.chat.completions.ChatCompletionCreateParams;
import com.openai.models.chat.completions.ChatMessage;

public class Grok3miniDemo {
    public static void main(String[] args) {
        // 1. 创建客户端，关键步骤：改baseUrl
        OpenAIClient client = OpenAIHttpClient.builder()
                // 把你的Key放进来，去千聚后台申请
                .apiKey("sk-你从千聚获取的API_KEY")
                // 核心：把地址换成千聚的，不需要翻墙了
                .baseUrl("https://www.qianjuai.com/v1")
                .build();

        // 2. 构建请求 - 模型名直接填Grok3mini
        ChatCompletionCreateParams params = ChatCompletionCreateParams.builder()
                .model("grok-3-mini") // 这里填千聚支持的模型名，Grok3mini就是这个
                .addUserMessage("用Java写一个简单的单例模式")
                .maxTokens(1024)
                .build();

        // 3. 发送请求并获取回复

        String content = completion.choices().get(0).message().content().get();
        System.out.println(content);
    }
}

## 深入一点：流式输出加配置

在实际生产环境中，你肯定要用流式输出（Streaming）来提升用户体验，防止请求超时。下面这个示例演示了怎么用`streaming()`模式调Grok3mini：

java
import reactor.core.publisher.Flux;
import com.openai.models.chat.completions.ChatCompletionChunk;
import com.openai.models.chat.completions.ChatCompletionCreateParams;

public void streamGrok3mini(String prompt) {
    OpenAIClient client = OpenAIHttpClient.builder()
            .apiKey("sk-你的Key")
            .baseUrl("https://www.qianjuai.com/v1")
            .build();

    ChatCompletionCreateParams params = ChatCompletionCreateParams.builder()
            .model("grok-3-mini")
            .addUserMessage(prompt)
            .maxTokens(2048)
            .streaming(true) // 开启流式输出
            .build();

    Flux<ChatCompletionChunk> stream = client.chat().completions().createStreaming(params);
    stream.subscribe(chunk -> {
        String delta = chunk.choices().get(0).delta().content().orElse("");
        System.out.print(delta);
    });
}

这段代码你完全可以丢到Spring Boot的Controller里，前端就能看到逐字显示的效果，体验直接拉满。
