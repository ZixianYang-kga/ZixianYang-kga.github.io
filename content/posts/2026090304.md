### 完整代码示例（含错误处理、流式输出）

实际开发中，流式输出（Streaming）可以提升用户体验，让 AI 回答像打字一样逐个出现。下面是一个支持流式输出的版本：

javascript
import OpenAI from 'openai';

const client = new OpenAI({
  baseURL: 'https://www.qianjuai.com/v1',
  apiKey: '你的千聚API Key',
});

async function mainStream() {
  try {
    const stream = await client.chat.completions.create({
      model: 'qwen3-max',
      messages: [
        { role: 'user', content: '给我写一首关于春天的五言绝句。' },
      ],
      temperature: 0.7,
      max_tokens: 512,
      stream: true, // 开启流式输出
    });

    console.log('Qwen3-Max 流式回答：');
    for await (const chunk of stream) {
      // 提取增量内容
      const content = chunk.choices[0]?.delta?.content || '';
      process.stdout.write(content); // 实时打印
    }
    console.log('\n（流式输出结束）');
  } catch (error) {
    console.error('流式请求失败：', error);
  }
}

mainStream();


这个版本会在终端逐个 token 地输出诗句，效果非常接近你平时看到的 AI 打字效果。
