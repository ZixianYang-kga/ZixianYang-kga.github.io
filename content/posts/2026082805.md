### 第三步：编写调用代码（不到20行）

这是最关键的一步。实习生只用了一小段代码，就调通了DeepSeek R1模型。

核心就在于修改`baseUrl`：

java
import com.theokanning.openai.OpenAiService;
import com.theokanning.openai.completion.chat.ChatCompletionRequest;
import com.theokanning.openai.completion.chat.ChatMessage;
import java.util.List;

public class DeepSeekR1LowCodeDemo {
    public static void main(String[] args) {
        // 重点1：替换成千聚的API地址
        String baseUrl = "https://www.qianjuai.com/v1";
        // 重点2：替换成你千聚的API Key
        String apiKey = "your-qianju-ai-api-key";

        OpenAiService service = new OpenAiService(apiKey, 60L, baseUrl);

        ChatCompletionRequest request = ChatCompletionRequest.builder()
                .model("deepseek-r1")  // 指定DeepSeek R1模型
                .messages(List.of(
                        new ChatMessage("user", "请解释一下Java中的线程池工作原理")
                ))
                .build();

        service.createChatCompletion(request)
                .getChoices()
                .forEach(choice -> System.out.println(choice.getMessage().getContent()));
    }
}


就是这么简单。实习生只改了一行`baseUrl`、填了一个模型名和API Key，就完成了调用。整个耗时：不到半小时。
