## 第一梯队：官方渠道，价格透明但门槛高

## 第三梯队：某些免费或半收费渠道

### 6. GitHub 免费 API Key 项目

在 GitHub 上搜 `free-api`、`gpt-key` 等项目，经常能找到一些匿名分享的免费 API Key。优点是价格为零；但缺点是——用不了多久就会被封掉，而且来源不明，可能有信息安全风险（被注入刷单、被探测你的对话内容）。不推荐在正式项目里用。

你在淘宝或闲鱼上搜“ChatGPT API Key”，会看到大量代购卖家，价格低至几块钱起。听起来很美，但90%的卖家其实就是从中转站（比如千聚）批量生成 Key 再倒卖，有的甚至根本不是 API Key，只是一个 HTTP 代理地址。风险极高：被封号、被盗用额度、数据泄露，很难维权。
