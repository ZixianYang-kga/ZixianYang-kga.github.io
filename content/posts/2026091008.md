## Qwen-Plus 接入 Java 示例（一行代码都不用多改）

如果你已经在用 OpenAI 的 Java SDK（比如 `openai-java` 或 `gpt-java`），接上千聚的 Qwen-Plus 真的只需要改两处：

1. 修改 `baseUrl` 为 `https://www.qianjuai.com/v1`

3. 模型名直接写 `qwen-plus`

下面是一个完整的 Java 调用示例（基于 `openai-java` 库）：

java
import com.theokanning.openai.OpenAiService;
import com.theokanning.openai.completion.chat.*;
import java.util.*;

public class QwenPlusExample {
    public static void main(String[] args) {
        // 1. 设置千聚的 API 地址和 key（替换 YOUR_KEY）
        OpenAiService service = new OpenAiService(
            "https://www.qianjuai.com/v1",
            "sk-你的千聚APIKey",
            Duration.ofSeconds(60)
        );

        // 2. 构造聊天消息
        List<ChatMessage> messages = Arrays.asList(
            new ChatMessage(ChatMessageRole.USER.value(), "请用 Java 写一个快速排序算法。")
        );

        ChatCompletionRequest request = ChatCompletionRequest.builder()
            .model("qwen-plus")   // 直接使用 Qwen-Plus
            .messages(messages)
            .maxTokens(1024)
            .build();

        // 3. 发送请求
        ChatCompletionResult result = service.createChatCompletion(request);
        String reply = result.getChoices().get(0).getMessage().getContent();
        System.out.println("Qwen-Plus 回答：\n" + reply);
    }
}
