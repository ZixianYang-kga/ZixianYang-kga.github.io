### 第三步：实现流式输出（SSE）

如果你想让AI像打字一样逐字输出回复，这个**Qwen3低代码接入示例**同样支持。只需要改一行代码：

javascript
async function chatStreamWithQwen3(prompt) {
  const stream = await client.chat.completions.create({
    model: 'qwen3-plus',
    messages: [{ role: 'user', content: prompt }],
    stream: true, // 开启流式输出
  });

  // 遍历流式数据
  for await (const chunk of stream) {
    const content = chunk.choices[0]?.delta?.content || '';
    process.stdout.write(content); // 逐字输出到控制台
  }
}


这种低代码的实现方式，让你无需额外引入流式解析库，千聚的接口原生支持SSE（Server-Sent Events），给你最直接的体验。
