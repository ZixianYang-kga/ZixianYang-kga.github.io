## 第三步：写代码接入——就改一行

这是最让人舒服的部分。假设你之前用过 OpenAI 的 API，那你现在的代码几乎不用改。

### 传统方式（需要科学上网）

python
import openai

# 旧配置：需要代理环境
openai.api_key = "your_openai_key"
openai.api_base = "https://api.openai.com/v1"

response = openai.ChatCompletion.create(
    model="gpt-4",
    messages=[{"role": "user", "content": "你好"}]
)
